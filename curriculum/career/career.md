## 🎯 Interview Preparation

### Before Interview Checklist
```
✅ Resume updated and reviewed
✅ LinkedIn profile optimized
✅ GitHub active with projects
✅ DSA practiced (50+ problems)
✅ System Design basic concepts
✅ Company research done
✅ Questions for interviewer ready
```

### Common HR Questions
1. **Tell me about yourself** - 2 min summary of your journey
2. **Why this company?** - Research and show excitement
3. **Your strengths/weaknesses?** - Honest but improvement focused
4. **Where do you see yourself in 5 years?** - Growth oriented
5. **Why did you leave your last job?** - Positive framing
6. **Salary expectations?** - Market research based

### Technical Interview Formats
```
FAANG-style:
├── DSA (45-60 min) - LeetCode Medium/Hard
├── System Design (45-60 min) - Design Twitter/Uber
└── Behavioral (30-45 min) - STAR method

Startup-style:
├── Project deep-dive (30 min)
├── Practical coding (30 min)
└── Culture fit (30 min)

Freelance:
├── Portfolio review
├── Client communication test
└── Technical assessment
```

### STAR Method
```
S - Situation (Context)
T - Task (Your responsibility)
A - Action (What you did)
R - Result (Quantifiable outcome)

Example:
"S: Our app was loading in 8 seconds
T: I needed to improve performance
A: Implemented lazy loading, image optimization, and caching
R: Reduced load time to 2 seconds (75% improvement)"
```
